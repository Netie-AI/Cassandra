## HOW TO USE THIS FILE
The English block is canonical (generated by the report pipeline). The ZH and MS
blocks are the target quality for the cheap-model translation + better-model gate.
The "TRANSLATOR REASONING" notes explain WHY each choice was made — this is what the
gating model checks against, and what a learning model should internalize.

Hard rules, every language:
1. NO em-dashes (—). Use comma, colon, period, or a new sentence. Mechanical lint enforces this.
2. Natural register, not machine-translation stiffness. Read it aloud; if a native
   finance reader would not say it, rewrite.
3. Finance terms use the conventional local rendering, not literal word-for-word.
4. Numbers, tickers, and dates stay in source form (38.8, MU, March 14 2000) but the
   PROSE around them is fully translated, including the date when written out in words.
5. EVERYTHING translates: headline, subhead, body, section headers, captions, table
   labels, analog notes, footer. If a string is visible and generated, it is translated.
   The ONLY untranslated items are proper nouns (Cassandra, WSJ) and tickers.

—— TRANSLATOR REASONING (ZH) ——
- "awareness band" → 「意识」区间 with corner brackets, the Chinese convention for a

- "swings harder once pushed" → 「摆动得更猛」 natural spoken Chinese, not the literal

- ZERO em-dashes. The EN "—" became 板块标题 colons or new clauses.
- "choppy to lower" → 震荡偏弱, the standard CN market-desk phrase, not a literal gloss.

## PIPELINE INSTRUCTION (how the model uses this)

For each new edition:
1. Generate EN body (canonical).
2. Cheap model translates EN → ZH and EN → MS, INSTRUCTED to match the register,
   metaphor-handling, and term conventions shown above. Pass this file as the example.
3. Better model gates each translation, checking ONLY:
   a. No em-dash present (also caught by mechanical lint as backstop).
   b. Register reads natural to a native finance reader (not MT-stiff).
   c. Finance terms use conventional local forms (倍 / KDNK / band names consistent).
   d. NOTHING left in English except proper nouns and tickers.
   If any check fails, return to step 2 with the specific failure noted.
4. Store all 3 bodies under (asof, lang). Toggle swaps stored body. No mixing, ever.

The dateline date ALSO translates its prose ("Today feels like" → 今日情势令人想起 /
Hari ini terasa seperti) while keeping "March 14, 2000" / "14 Mac 2000" formatted
per locale. This was the specific bug: the date line stayed English. It must not.
